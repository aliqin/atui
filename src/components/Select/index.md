## API
`Select`组件包含子组件 `Option`

### Select

属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
value | 选中的值，当设置为多选时用Array | String / Array | |
placeholder | 占位提示 | String | |
multiple | 是否多选 | Boolean |  false |
tags | 是否输入变Tag | Boolean | false |
disabled | 是否禁用 | Boolean | false |

### Option
属性 | 说明 | 类型 | 默认值
-----|-----|-----|------
value | 选项的值 | String | |
disabled | 是否禁用 | Boolean | |
